WRScaler

A jQuery plugin that applies a CSS3 scale transform to an element to resize it below a specified window breakpoint width.

Download this project as a .zip file Download this project as a tar.gz file

About WRScaler

Useful when natural HTML element scaling doesn't do enough. For instance you might have a component or complex layout that relies on absolute positioning. Wrapped elements will have a scale transform dynamically applied on window resize when the window width falls below a designated threshold value. The scale will be in direct proportion of the window width to the threshold width with respect to the origninal non-scaled width of the element.
